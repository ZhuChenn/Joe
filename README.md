## Quick Start

Just open `index.html` in your browser - no build tools required!

Or serve it locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000`

## Customization

### Update Contract Address
Edit the contract address in `index.html`:
```html
<code id="contract">YOUR_CONTRACT_ADDRESS_HERE</code>
```

### Update Social Links
Find the social cards in the community section and update the `href` attributes.

### Update Stats
Modify the hero stats section with real market data.
